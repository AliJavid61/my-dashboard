# Opofinance KYC Platform Dashboard

A frontend-only dashboard for the KYC Team Manager at Opofinance forex brokerage. No backend, no build tools — just open and run.

## Description

This dashboard provides a centralized interface for managing KYC (Know Your Customer) operations, including document review queues, withdrawal requests, team workload monitoring, and risk flag tracking.

## Features

- **Login Page** — Simple authentication screen to access the dashboard
- **KYC Queue Management** — Review and process pending KYC document submissions
- **Withdrawal Review** — Evaluate and approve/reject withdrawal requests
- **Team Workload Tracking** — Monitor task distribution and progress across analysts
- **Risk Flags** — View and manage flagged accounts and suspicious activity
- **Dark/Light Mode Toggle** — Switch between dark and light themes

## How to Run

No server, no install, no build step required.

1. Clone or download this repository
2. Open `index.html` in any modern browser

That's it.

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
